Unemployment in substates of Germany

Project information:

Project for the course "Geoinformation in Society" in the winter semester 2017/2018 at ifgi, University of MÃ¼nster.
The goal is to create an application which consumes and visualizes linked open data (LOD). The application should ease the understanding of the data used for the visualisation.
For this project, we chose to visualize unemployment data in Germany on the level of federal states. Moreover we want to add different data sets and build LOD so a connection of this data to unemployment data can be seen.
